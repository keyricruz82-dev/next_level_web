Guía visual del usuario

Manual de usuario de Next Level Producciones

Esta guía está pensada para ayudar al dueño del sitio a entender, mostrar y explicar el funcionamiento del proyecto de forma clara, visual y práctica.

Objetivo: traducir funciones técnicas en instrucciones sencillas para que cualquier persona pueda usar el sitio sin dificultad.

1. Cómo abrir la página

  1. Abre el navegador de tu preferencia.
  2. Entra a la dirección local del proyecto o abre la carpeta donde está el sitio.
  3. Selecciona la página principal para comenzar.

2. Cómo moverte por el sitio

  1. Usa la barra superior para ir a Inicio, Servicios, Eventos y Contacto.
  2. Haz clic en cualquier opción para cargar la sección correspondiente.
  3. Si estás en celular, usa el botón del menú para ver todas las opciones.

3. Cómo usar el menú móvil

  1. En pantallas pequeñas aparecerá el botón de menú.
  2. Presiona el botón para desplegar las opciones.
  3. Selecciona la sección que deseas ver y el menú se cerrará automáticamente.

4. Cómo cambiar el tema

  1. Busca el botón de cambio de tema en la parte superior.
  2. Haz clic y cambia entre modo claro y oscuro.
  3. El sitio conservará esa preferencia al abrirlo de nuevo.

6. Cómo ver los servicios

  1. Entra a la sección Servicios.
  2. Observa las tarjetas con fotografía, video y producción.
  3. Si quieres saber más, revisa el contenido relacionado a cada servicio.

7. Cómo explorar la galería

  1. Revisa la galería principal de la página de inicio.
  2. Haz clic sobre las imágenes para ver el detalle del contenido.
  3. En la sección de eventos podrás ampliar la información de cada producción.

8. Cómo usar el contacto y WhatsApp

  1. Desplázate hacia la parte inferior de cualquier página.
  2. Usa la sección de contacto o el botón flotante de WhatsApp.
  3. Desde ahí puedes comunicarte con el equipo de forma rápida.

9. Cómo aprovechar el diseño responsive

  1. Prueba abrir el sitio en computadora, tablet o celular.
  2. Reduce o amplia la ventana para ver cómo se adapta el contenido.
  3. El texto, menú e imágenes cambian de tamaño de forma automática para mejorar la experiencia.

10. Consejos para el usuario

  • Usa el menú móvil cuando estés en un teléfono.
  • Explora primero la sección de eventos y servicios para conocer el trabajo.
  • Si el sitio se ve muy cargado, espera unos segundos y recarga la página.
  • Las imágenes ayudan a identificar cada sección de forma rápida.

Este manual está diseñado para ser claro, visual y útil para el dueño del sitio y cualquier persona que lo administre o lo presente.
